8. port

```minicom 
port <port>
```

Используйте эту команду, чтобы указать порт, используемый сервером REST API.

```minicom
no port
```

Используйте эту команду, чтобы использовать порт по умолчанию (8082) для сервера REST API.

#### Параметры

| Имя  | Тип       | Описание |
| ---- | --------- | -------- |
| port | <0-65535> | Порт     |
