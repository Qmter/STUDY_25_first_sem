### **üîπ –£–ø—Ä–∞–≤–ª–µ–Ω–∏–µ —Ü–∏–∫–ª–∞–º–∏ (break, continue, else)**

**break**¬†- –¥–æ—Å—Ä–æ—á–Ω—ã–π –≤—ã—Ö–æ–¥ –∏–∑ —Ü–∏–∫–ª–∞:

```python
for num in range(10):
    if num == 5:
        break
    print(num)  # 0 1 2 3 4
```

**continue**¬†- –ø–µ—Ä–µ—Ö–æ–¥ –∫ —Å–ª–µ–¥—É—é—â–µ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏:

```python

    if num == 2:
        continue
    print(num)  # 0 1 3 4
```

**else**¬†–≤ —Ü–∏–∫–ª–∞—Ö - –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è, –µ—Å–ª–∏ —Ü–∏–∫–ª –∑–∞–≤–µ—Ä—à–∏–ª—Å—è –µ—Å—Ç–µ—Å—Ç–≤–µ–Ω–Ω–æ (–±–µ–∑ break):

### **üîπ –ü—Ä–∏–º–µ—Ä—ã –∫–æ–º–±–∏–Ω–∞—Ü–∏–π**

    print(f"{num} –ø—Ä–æ—Å—Ç–æ–µ")
